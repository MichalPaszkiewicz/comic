Adult marketing

You have a very good product, that I'm sure adults will love.
It is very well thought out, it has great aesthetics and it solves most everyday problems.
This product could change the world.
But unfortunately, no serious adult will buy it.
HUH? What's wrong with it?
You need to package it in a SERIOUS box.
